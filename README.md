# unitypm
Unity Package Manager allows you to manage local dependences for Unity

## CLI Installation
```
npm i -g @forjagames/unitypm
```

## Create a new package
In your Assets folder
```
unitypm init
```

Then
```
unitypm publish
```

## Install the package
In a folder
```
unitypm install <my package name>
```

Example
```
unitypm install code-in-unity
```
