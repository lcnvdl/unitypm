# unitypm
Unity Package Manager allows you to manage local dependences for Unity

## CLI Installation
```
npm i -g @forjagames/unitypm
```

## Create a new package
In your Assets folder
```
upm init
```

Then
```
upm publish
```

## Install the package
In a folder
```
upm install <my package name>
```

Example
```
upm install code-in-unity
```
